O4AA Config Clipboard

One-click config extraction from the Okta admin console

Drag this button to your bookmarks bar:

O4AA Config Clipboard

How to Use

1
Go to your OIDC App page in Okta Admin → click the bookmarklet. It grabs the Client ID and derives the Issuer URLs.
2
Go to the Custom Auth Server page → click the bookmarklet. It grabs the server ID and derives all JAG + Resource endpoint URLs.
3
For manual values (Client Secret, JWK, Key ID): highlight the text on the page, then click the bookmarklet. It prompts you to assign the selection to a field.
4
Go to the demo app at /settings → click "Paste Config JSON". All collected values fill in automatically.
How it works: Each click accumulates values in sessionStorage (cleared when you close the tab). The bookmarklet auto-detects which Okta admin page you're on, extracts what it can from the DOM, and merges with previously collected values. The JSON is copied to your clipboard on every click so you always have the latest state.
